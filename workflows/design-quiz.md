# Workflow: design-quiz

Ask a short quiz to unblock implementation.

## Rules

- Maximum 5 questions.
- No generic questions.
- Each question must affect implementation.
- Use multiple choice when possible.
- If the answer is obvious from the project, infer it.

## Default quiz

1. What are we improving?
   - homepage
   - pricing page
   - dashboard
   - specific component
   - whole app audit only

2. What reference contract should be used?
   - existing contract
   - new URL
   - no reference, improve internally

3. How aggressive should the change be?
   - light polish
   - balanced redesign
   - strong visual direction

4. What must not change?
   - copy
   - layout
   - brand colours
   - business logic
   - all of the above

5. Should motion be applied?
   - no
   - minimal
   - moderate
   - only if already present

## Output

After the user answers, produce a short implementation plan and proceed only with safe, scoped changes.
